Description:
Abstract:
Updated database of the Federal Railroad Administration's (FRA) Amtrak Station database. This database is a geographic data set containing Amtrak intercity railroad passenger terminals in the United States and Canada. Attribute data include services and passenger amenities provided at the station.
Purpose:
The data set provides an updated version of the location and attribute information for all Amtrak stations in the United States and Canada. This data is used for national and network analysis applications.

Data_Quality_Information:
Logical_Consistency_Report:
All points records are represented by a single coordinate pair and a unique feature identifier; the Amtrak Station Code (stored in the STNCODE field). Attribute data can be matched to specific point records using this identifier.
Completeness_Report:
The database contains all the Amtrak thruway bus and rail stations from the Amtrak website and Timetables in the US and Canada as of March 2009. As new stations are formed or if they move, this database should be updated accordingly.

Process_Step:
Process_Description:
FRA's Amtrak Database was copied and the attributes for each station was compared with the Amtrak webiste. If there were any dissimilarities, the attribute table was changed. If an address of a station changed, the location of the station was observed to determine if the point had to be moved. If a station location had to be moved, it was done manually. Newly added stations were first but into a seperate shapefile (a shapefile for each geographic ragion) and were added manually by cross-referencing their location with road atlases. Then the shapefiles were added to the database.
